Robotics & AI Research

Technical projects demonstrating deep expertise in robot autonomy, perception, and learning systems.

Robotic Path Planning & Manipulation

Real-time demonstration of motion planning algorithms and manipulation control in simulation environment.

Implementation Framework

  • Perception Module: Point cloud processing and object detection using PCL and OpenCV
  • Planning Layer: RRT* path planning with collision avoidance in constrained workspace
  • Control System: Joint-space trajectory optimization with inverse kinematics solver
  • Simulation: CoppeliaSim environment with 6-DOF manipulator and dynamic obstacles
ROS 2 Python C++ MoveIt2 PCL

Vision-Guided Suturing Assistant

Autonomous needle hand-off and suture tracking with real-time perception feedback in a dual-arm surgical setup.

Implementation Framework

  • Vision Stack: Stereo depth fusion with YOLOv8-based instrument segmentation at 40 FPS
  • Task Planner: Behavior tree coordinating needle hand-off and collision-aware arm sequencing
  • Force Feedback: Admittance control tuned for delicate tissue interaction using impedance models
  • Evaluation: Reproduced 20 suturing trials inside a da Vinci Research Kit simulator
ROS 2 TensorFlow C++ OpenCV dVRK

Collaborative Mobile Manipulation Fleet

Coordinated warehouse robots combining LiDAR SLAM navigation with shared manipulation tasks under human supervision.

Implementation Framework

  • Perception: Multi-sensor fusion of 3D LiDAR and RGB-D cameras for dynamic obstacle tracking
  • Fleet Coordination: Decentralized task allocator with consensus messaging over ROS 2 DDS
  • Manipulation: Isaac Sim-trained grasp policies deployed via on-board Jetson Orin accelerators
  • Monitoring: Web-based operator console with latency-aware telemetry and safety interlocks
ROS 2 Isaac Sim CUDA LiDAR SLAM Jetson Orin
