Robotics & AI Research

Robotic Path Planning & Manipulation

Real-time demonstration of motion planning algorithms and manipulation control in simulation environment.

Implementation Framework

  • Perception Module: Point cloud processing and object detection using PCL and OpenCV
  • Planning Layer: RRT* path planning with collision avoidance in constrained workspace
  • Control System: Joint-space trajectory optimization with inverse kinematics solver
  • Simulation: CoppeliaSim environment with 6-DOF manipulator and dynamic obstacles
ROS 2 Python C++ MoveIt2 PCL

Object Detection-Driven Robot Arm Tracking

Real-time, low-latency target following for robotic arms powered by a custom object detection pipeline.

Implementation Framework

  • Detection Core: Deployed a customized object detection model to recognize arm targets at high speed.
  • Latency Optimization: Tuned the inference and control pipeline to minimize end-to-end delay for responsive tracking.
  • Tracking Deployment: Realized precise motion tracking inside a virtual environment for closed-loop evaluation.
ROS 2 YOLOv8 TensorRT OpenCV C++

Object Pose Estimation & Stacking via Eye-in-Hand Camera

6D object pose estimation with an eye-to-hand setup enabling automated stacking tasks.

Implementation Framework

  • Perception: Achieved 6D pose estimation using an eye-to-hand camera configuration.
  • Control: Drove the robotic arm to complete precise stacking based on the predicted poses.
  • Optimization: Jointly refined estimation latency and arm motion accuracy for reliable execution.
ROS 2 6D Pose Eye-to-Hand CoppeliaSim C++

Monocular Endoscopic Depth Estimation (Depth Anything)

Millimeter-precision depth estimation in endoscopic scenes using Depth Anything with optical-flow refinement.

Implementation Framework

  • Perception: Fuse Depth Anything predictions with optical-flow refinement to stabilize depth.
  • Calibration: Benchmark against surgical calibration phantoms for quantitative accuracy.
  • Integration: Provide reliable depth cues for downstream surgical scene understanding.
